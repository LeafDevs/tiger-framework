## Getting Started

### Prerequisites
- Bun (v1.0 or higher)

### Installation

1. Clone the repository from GitHub
```bash
git clone https://github.com/leafdevs/tiger-framework.git
cd tiger
```

2. Install dependencies
```bash
bun install
```

3. Build the project
```bash
bun run build
```

4. Start the development server
```bash
bun run start
```

5. Open your browser and navigate to `http://localhost:3000`
